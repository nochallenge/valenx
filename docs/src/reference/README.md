# Reference

Reference material for users who already know what they want to
do. The current chapters:

- [CLI reference](./cli.md) — every headless tool Valenx ships,
  with usage, flags, exit codes, and a CI-pipeline recipe that
  chains them.

The longer-form reference (file formats, settings, keyboard
shortcuts) is planned but not yet written. Track progress in the
top-level [STATUS.md](https://github.com/nochallenge/valenx/blob/master/STATUS.md).
