# Contributing

This chapter mirrors the root
[CONTRIBUTING.md](https://github.com/nochallenge/valenx/blob/main/CONTRIBUTING.md)
for in-app readers. For the authoritative, always-up-to-date version,
see the file in the repo.

---

## TL;DR

1. Read the [ROADMAP](https://github.com/nochallenge/valenx/blob/main/ROADMAP.md)
   and pick something that interests you
2. For bug fixes / small features: open a PR directly
3. For bigger changes: file an RFC first
4. Be respectful — see the
   [Code of Conduct](https://github.com/nochallenge/valenx/blob/main/CODE_OF_CONDUCT.md)
5. All contributions are under Apache 2.0

Full details, build instructions, commit conventions, and the RFC
process live in the root repository.
