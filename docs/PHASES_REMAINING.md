## Tier system

- **T1 (Achievable, ≤1 session each):** Maps onto existing primitives. Implementation is wiring + math, no research.
- **T2 (Multi-session, ≤2 weeks of focused work):** Needs new code paths but well-understood algorithms.
- **T3 (Major project, 1-6 months each):** Real research/engineering — BRep math, format reverse engineering, etc.
- **T4 (Out of scope or upstream-dependent):** Either needs external project to ship something first, or genuinely is not this project's domain.

> **2026-05-22 — `valenx-cfd-native` commercial-depth: k-ω SST + geometric multigrid + Ghia validation.** The 2-D incompressible SIMPLE solver (`valenx-cfd-native`) graduated the three named gaps to commercial-depth 2-D-CFD parity. **(A) Menter k-ω SST turbulence** (`turbulence.rs`) — the external-aero standard alongside the existing k-ε: two transport equations for `k` and the specific dissipation `ω`, the `F₁` near-wall ↔ free-stream blending function, the `F₂` blending inside the SST eddy-viscosity limiter `ν_t = a₁·k / max(a₁·ω, S·F₂)`, the `(1−F₁)` cross-diffusion term, the production limiter `min(P_k, 10·β*·ρ·k·ω)` at stagnation points, and Menter's near-wall `ω` boundary value `ω_wall = max(6ν/(β₁·y²), u_τ/(√β*·κ·y))`. Standard Menter 1994 constants — `β*=0.09`, `a₁=0.31`, set-1 `(σ_k, σ_ω, β) = (0.85, 0.5, 0.075)`, set-2 `(σ_k, σ_ω, β) = (1.0, 0.856, 0.0828)`, `γ` from Menter's `γ_i = β_i/β* − σ_ωi·κ²/√β*`. A wall-distance helper (closed form for rectangular domains) feeds `F₁` / `F₂`; a fully-developed-channel SST driver mirrors the k-ε one for verification. The new [`EffectiveViscosity`] enum lets the SIMPLE driver consume Laminar / k-ε / SST through one `solve_simple_with(...)` call; the cell-centred eddy viscosity is averaged onto each momentum face (`ν_eff = ν + ν_t`). **(B) Geometric-multigrid V-cycle pressure solver** (`multigrid.rs`) — a real V-cycle on the staggered grid: weighted-Jacobi smoother at `ω = 2/3` (the textbook 5-point-Laplacian optimum), full-weighting 4:1 cell-aggregation restriction, bilinear-interpolation prolongation, cell-aggregation coarse operator with the correct `1/4` cell-size-ratio scaling for cell-centred FV-Galerkin coarsening of a 5-point stencil. Wired through a new `SimpleControls.pressure_solver: PressurePoissonSolver::{Sor, Multigrid}` selector; SOR stays the default + fallback. Measured per-cycle residual reduction is **essentially grid-independent** — `0.176` on 32², `0.180` on 64², `0.181` on 128² (textbook multigrid range 0.1–0.3), exactly the property SOR fails to deliver. **(C) Ghia 1982 / Poiseuille / BFS published-reference validation suite** (`benchmark.rs`).
