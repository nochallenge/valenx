# Rejected / withdrawn RFCs

This directory holds RFCs that were filed and **not** accepted, kept on
the record for posterity.

## Why archive rejected RFCs?

1. **Future RFCs can reference them.** "This was already considered in
   `rejected/0009-some-idea.md`; here's what's changed since" saves
   everyone the rediscovery.
2. **Rationale is preserved.** Anyone reading in 2030 can see *why* an
   idea was turned down, not just that it was.
3. **Withdrawn RFCs aren't failures.** Sometimes the author realizes
   there's a better design; sometimes upstream changes moot the
   proposal; sometimes timing is wrong. Keeping them here says that
   the process is working, not that the author was wrong.

## Format

Each file is the original RFC with one additional header block at the
top:

```markdown
> **Status:** Rejected (or Withdrawn)
> **Decision date:** YYYY-MM-DD
> **Discussion PR:** #nnn
> **Reason:** One-paragraph summary of why.
```

The original body is preserved verbatim — no rewriting history.
